	CSS3 Responsive Content Tabs for WordPress.

== Description ==


Tabs is pure HTML & CSS3 responsive Tabs grid for WordPress. By this plugin you can display unlimited tabs grid at same page via short-code. super responsive for mobile view will convert as accordion.


### Tabs by http://paratheme.com

<br />


<strong>Plugin Features</strong><br />

* Fully responsive and mobile ready.
* Unlimited Tabs anywhere.
* Use via short-code.
* Different Theme.
* Easy input field for Tabs content.
* Background Image for Tabs area.
* Custom Active tabs background color.
* Custom default background color.
* Custom font color and size for Tabs header.
* Custom font color and size for Tabs content.




== Installation ==

1. Install as regular WordPress plugin.<br />
2. Go your plugin setting via WordPress Dashboard and find "<strong>Tabs</strong>" activate it.<br />

After activate plugin you will see "Tabs" menu at left side on WordPress dashboard click "New Tabs" and use the options field "Tabs Options"<br />

<br />
<strong>How to use on page or post</strong><br />
When Tabs options setup done please publish Tabs as like post or page<br />

and then copy shortcode from top of <strong>Tabs Options</strong> `[tabs  id="1234" ]`<br />

then paste this shortcode anywhere in your page to display Tabs<br />
